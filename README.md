# Installation
1. Download Conda (https://conda.io/miniconda.html or https://www.anaconda.com/download)
2. Set up a Conda environment:
    - `conda env create -f environment.yml`
3. Activate the environment:
    - `source activate sassyvoice`
4. Hack
